# React Frontend Agent Entry Point

Start with `.agents/AGENTS.md`.

Use that file as the main brain, then load only the relevant documents for the task instead of reading the whole folder every time.

Default expectations:

- Preserve the existing UI patterns, design system, and architecture.
- Keep components small, composable, and easy to reason about.
- Treat accessibility, responsive behavior, and performance as part of correctness.
- Reuse existing state, data-fetching, routing, and testing patterns before adding new ones.
- Add or update tests for meaningful behavior changes.
- Run the final checklist before finishing.
