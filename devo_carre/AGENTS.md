# Spring Boot Backend Agent Entry Point

Start with `.agents/AGENTS.md`.

Use that file as the main brain, then load only the relevant documents for the task instead of reading the whole folder every time.

Default expectations:

- Preserve the project architecture and existing conventions.
- Prefer constructor injection, clear boundaries, and explicit transactions.
- Treat validation, security, pagination, N+1 risk, and tests as part of correctness.
- Reuse the project’s patterns for persistence, API design, and integrations.
- Run the final checklist before finishing.
